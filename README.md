# Todo App

Next.js（App Router）と Supabase を使用して作成した、  
**認証付きのシンプルなTodo管理アプリ**です。

未経験からWebアプリ開発を学ぶ中で、  
「ログイン → CRUD → デプロイ」まで一通り経験することを目的に制作しました。

---

## Demo
http://localhost:3000/login

---

## Features

- ユーザー登録 / ログイン
- Todoの新規作成
- Todo一覧表示
- Todo詳細表示
- Todo編集
- Todo削除
- ログアウト機能

---

## Tech Stack

- Next.js 16
- TypeScript
- Supabase
- Vercel
