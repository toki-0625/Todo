# オリジナルプロダクト概要

## 一言サービスコンセプト
**「今日やるべきことが、ひと目でわかるTodo管理」**

---

## 誰のどんな課題を解決するのか？
- 学習中の初学者や、日常のタスク管理が苦手な人
- 高機能すぎるTodoアプリは機能全てを使いこなせず、継続できない

---

## なぜそれを解決したいのか？
- 自分自身が学習や日常生活で、何から手を付ければよいか分からなくなることが多かったため
- 必要最低限＋少し便利な機能に絞ったTodoアプリを作りたいと考えたため

---

## どうやって解決するのか？
- タスクに完了状態・優先度・期限といった情報を持たせる
- 色分けや強調表示によって、タスクの状態を直感的に判断できるようにする
- React + TypeScript を用いて、状態管理をシンプルに実装する

---

## 機能要件

### 基本機能
- タスクの追加・削除・編集
- タスクの完了 / 未完了切り替え
- 入力欄が空のときは追加できない
- タスクが0件のとき「タスクがありません」と表示
- 最新のタスクを上に表示
- 未完了 / 完了タスクの件数表示

### 管理・操作機能
- 完了タスクに取り消し線を表示
- 完了タスクの文字色を変更（グレーなど）
- チェックを入れると完了タスクを下に移動
- 作成日時の表示
- タスク検索機能
- 完了タスクの一括削除
- 完了 / 未完了の一括切り替え
- 直前の削除を元に戻す
- 優先度（高・中・低）の設定
- 優先度ごとの色分け表示
- 今日が期限のタスクを強調表示
- 期限切れタスクを赤色表示

---

## 非機能要件

- **操作性**
  - 初心者でも迷わず使えるシンプルなUI

- **永続性**
  - ページをリロードしてもデータが保持される

- **開発環境**
  - React + TypeScript
